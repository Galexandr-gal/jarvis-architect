# Бренд-бук «Робот Мойка» — цвета и шрифты

> Источник: `knowledge/brendbuk-avtomoyka.pdf` (35 МБ, 18 страниц). Точные цвета — стр. 7, шрифты — стр. 6. Картинки этих двух страниц лежат рядом: `brendbuk-cveta.png`, `brendbuk-shrifty.png` — не нужно пересобирать PDF, чтобы свериться.

## Фирменные цвета (стр. 7)

| Роль | CMYK | RGB | HEX |
|---|---|---|---|
| Акцент (оранжевый) | 13, 38, 100, 0 | 208, 157, 66 | `#D09D42` |
| Тёмный (антрацит) | 0, 0, 0, 80 | 20, 20, 20 | `#141414` |
| Светлый (кремовый) | 1, 1, 7, 0 | 254, 250, 237 | `#FEFAED` |

**Важно:** кремовый (`#FEFAED`) в бренд-буке везде используется как *фон* (светлая футболка, светлая плашка) с тёмным текстом поверх — никогда как цвет текста на белом/светлом фоне. Именно этот кремовый цвет применён как цвет текста в карточках тарифов на сайте роботмойка.kz — отсюда нечитаемость (см. `memory/2026-07-16.md`, аудит сайта).

## Фирменные шрифты (стр. 6)

| Назначение | Шрифт |
|---|---|
| Рекламные баннеры (только акцент, коротко) | TT Norms Heavy |
| Заголовки, акцентные надписи | TT Norms Bold |
| Текстовые блоки, контакты, описания | Bahnschrift |

## Как обновить

Если бренд-бук изменится — заменить `brendbuk-avtomoyka.pdf` и пересобрать нужные страницы в PNG через встроенный Windows API (`Windows.Data.Pdf`), т.к. на машине нет poppler/imagemagick, а исходящий HTTPS из Bash недоступен, скачать их тоже нельзя. Рабочий PowerShell-скрипт для рендера страниц PDF → PNG отработан в сессии 2026-07-16, при необходимости можно восстановить логику через WinRT `PdfDocument`/`PdfPage.RenderToStreamAsync`.
